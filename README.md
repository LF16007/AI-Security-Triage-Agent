# 🛡️ AI-Security-Triage-Agent
**Status:** MVP - Privacy-First Local Engine 

### 🚀 Project Overview
An automated security triage system designed for high-compliance environments (London Fintech/Banking). This agent monitors logs in real-time and provides instant risk assessment without data egress.

### 🛠️ Key Features
*   **Privacy-First Architecture:** Uses a local logic engine to ensure sensitive logs never leave the local environment (UK GDPR compliant).
*   **Fintech Impact Mapping:** Automatically maps threats to FCA/PRA Operational Resilience standards.
*   **Structured Reporting:** Generates `triage_report.json` for seamless ingestion into enterprise SIEM tools.

### 📈 Market Value
This tool automates the "Level 1" analyst triage process, reducing manual log review time by an estimated 80%.
